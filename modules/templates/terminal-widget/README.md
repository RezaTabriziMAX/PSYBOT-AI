# terminal-widget (Template)

A React widget template aligned with the Nuttoo Fallout-terminal aesthetic.

Includes:
- Monospace layout with scanline/noise hooks
- Minimal command input + output stream pattern
- Pluggable command handlers

This is a template; you should adapt styling to your host web app.
